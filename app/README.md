# pytorch-flask-app
Simple app with flask and torch for digit classification

<h2>Libraries Needed</h2>
<ul>
<li>pytorch</li>
<li>requests</li>
<li>flask</li>
</ul>
<b>How to run</b>
<ul>
<li>export FLASK_APP=main.py</li>
<li>export FLASK_ENV=development (for hot reload,required only during development)</li>
<li>flask run</li>
</ul>
